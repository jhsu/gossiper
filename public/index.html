Gossiper

A playful Slack app that listens in-channel and sends private AI-generated gossip to opted-in teammates.

Install to Slack Health check

Private reactions

Gossiper sends ephemeral, private commentary instead of posting publicly in the channel.

Workspace-aware

Each Slack workspace installs independently with its own bot token and configuration.

Opt-in controls

Whitelist channels and recipients so gossip only goes where you want it to.

/gossip-channel Whitelist the current channel for gossip. Add off to disable it.
/gossip-me Opt yourself in to receive gossip. Add off to opt out.
